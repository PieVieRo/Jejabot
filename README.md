### Jejabot
Statystyki kont na [jeja.pl](https://jeja.pl).

### Komendy
```
j!pd - podaje PD użytkownika
j!link - podaje link do profilu (alias profil)
j!avek - podaje profilowe konta
j!strzalki - pokazuje ilość strzałek w górę
```
